# 🧩 全屋定制风格选择：权威数据与典型示意图

> 用于记录全屋定制的风格选择与材料认知。图片为风格示意图（非实拍），便于理解配色与材质氛围。

---

## 一、先确定 3 个“风格底座”

1. **采光与空间尺度**：采光弱、层高低 → 低对比浅色；采光足、层高高 → 可用深色与木色。
2. **收纳与家务习惯**：重收纳 → 平板门+隐藏拉手+整墙柜；爱展示 → 开放格与展示柜比例提高。
3. **预算与维护**：造型越多、拼接越复杂，成本与维护难度越高。

## 四、材料与工艺科普（与风格落地强相关）

### 1) 板材常识

| 板材             | 特点               | 适合位置         |
| ---------------- | ------------------ | ---------------- |
| 多层板（胶合板） | 稳定性好、耐潮性强 | 厨卫柜体、地台   |
| 颗粒板           | 稳定性好、性价比高 | 大面积柜体、衣柜 |
| 中纤板（MDF）    | 便于造型与烤漆     | 造型门板、弧形   |
| 实木/指接板      | 质感好、成本高     | 局部点缀         |

### 2) 封边与表面工艺

- **PUR 热熔封边**：耐水耐热好，适合厨卫与高湿区
- **激光封边**：缝隙小，观感更干净
- **肤感/抗指纹**：适合现代极简，但需注意耐磨等级

## 五、权威数据与标准（聚焦健康与可验证性）

### 1) 甲醛暴露限值（职业场景，仅作风险认知）

以下数据来自 NIOSH/OSHA 的职业暴露限值，用于帮助理解甲醛风险，不等同于住宅标准。

| 机构       | 指标             | 数值      |
| ---------- | ---------------- | --------- |
| NIOSH REL  | 8 小时 TWA       | 0.016 ppm |
| NIOSH REL  | 15 分钟 C        | 0.1 ppm   |
| OSHA PEL   | 8 小时 TWA       | 0.75 ppm  |
| OSHA PEL   | 15 分钟 ST       | 2 ppm     |
| NIOSH IDLH | 立即危害生命健康 | 20 ppm    |

- NIOSH 页面给出换算：**1 ppm = 1.23 mg/m3**。
- 住宅场景应远低于职业暴露限值，核心抓手是**选材+封边+通风**。

来源：NIOSH Pocket Guide – Formaldehyde
https://www.cdc.gov/niosh/npg/npgd0293.html

### 2) 复合木制品排放规则（美国 EPA）

EPA 对复合木制品（如硬木胶合板、MDF、刨花板）设定甲醛排放标准，并要求第三方认证与标签。选购时可要求：

- 第三方检测报告与认证信息
- 产品标签与批次号
- 供应链一致性说明

来源：EPA – Formaldehyde Emission Standards for Composite Wood Products
https://www.epa.gov/formaldehyde/formaldehyde-emission-standards-composite-wood-products

### 3) 国内标准索引（建议采购时核对报告）

- GB/T 39600-2021《人造板及其制品 甲醛释放量分级》
- GB 18580-2017《人造板及其制品中甲醛释放限量》
- GB/T 17657-2013《人造板及饰面人造板理化性能试验方法》

## 六、落地清单（选材/签合同前必问）

1. 板材检测报告与检测方法（是否为第三方、是否标注方法）
2. 封边工艺与胶型（是否 PUR/激光）
3. 五金品牌与型号（铰链/滑轨/反弹器）
4. 施工与安装周期（含补货与返工）
5. 交付后通风与除味建议
